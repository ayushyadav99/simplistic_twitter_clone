Howler- A Simplistic Twitter Clone
    Howler is a microblogging website built on the lines of Twitter. It includes the following features:
     
1. Create your own account.
2. Login using password.
3. Ability to micropost(howl)
4. Follow other people.
5. Get Howls(posts) of the people you follow on the homepage.
6. Page listing all howls.
7. Page listing all users.
8. Page showing your own profile containing howls posted by you.
11.View other users howls individually on their profile page.
10. Search function for finding information about user.
